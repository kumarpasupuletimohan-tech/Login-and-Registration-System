# Login-and-Registration-System
The Login and Registration System is a web-based authentication application developed using ASP.NET Core MVC. This system allows users to create accounts, log in securely, and access protected pages after authentication.
teraction, and MVC architecture using C# and Entity Framework Core.

This project is useful for understanding how authentication systems work in real-world web applications.

This project demonstrates the implementation of user authentication, database interaction, and MVC architecture using C# and Entity Framework Core.

 Features:

User Registration, User Login Authentication,Validation for incorrect login credentials,Database storage for user accounts, MVC architecture implementation, Secure data handling

 Technologies Used:
C#	(Backend programming),
ASP.NET Core MVC	(Web framework),
Entity Framework Core	(ORM for database operations),
Microsoft SQL Server	(Database),
HTML / CSS / Razor Views	(Frontend),
Microsoft Visual Studio	(Development environment)
